# Vendored fonts

Used by the vault reader (`site/reader.ts`). Copied into the reader output so it
renders with no network requests.

- **Newsreader** — SIL Open Font License 1.1. https://fonts.google.com/specimen/Newsreader
- **Spline Sans Mono** — SIL Open Font License 1.1. https://fonts.google.com/specimen/Spline+Sans+Mono

woff2 subsets fetched from Google Fonts; `../fonts.css` carries the `@font-face`
rules (localized to these files). Both families are OFL — free to bundle and
redistribute; keep this attribution.
